# 🏛️ Campus Square

**The All-in-One Verified Campus Ecosystem**
Replacing chaotic WhatsApp & Telegram groups with a structured, domain-verified platform for university students.

## 📖 About The Project

**Campus Square** is a comprehensive, mobile-first ecosystem designed exclusively for university and college campuses. Built with Flutter, it tackles the fragmentation of student life by consolidating notices, peer-to-peer commerce, academic resources, and community trust into a single, seamless application.

Access is strictly governed by domain-locked authentication (e.g., `@institute.ac.in`), ensuring a secure, verified environment free from spam and external noise.

## ✨ Core Modules & Features

The architecture is divided into four distinct, interconnected quadrants:

1. 📢 **The Square (Community Hub)**
   **Real-Time Notice Board**: Official announcements and student-driven notices with global push notifications.
   **Smart Roommate Finder**: Algorithmic matching based on structured lifestyle preferences (Diet, Sleep Schedule, Study Habits).
   **Lost & Found**: Geotagged and image-supported tracking for lost items around campus.
   **Ride Pooling**: Peer-to-peer transit coordination for sharing cabs to transit hubs or city centers.

2. 🛒 **The Bazaar (Marketplace)**
   **Verified Buy/Sell/Rent**: A localized marketplace exclusively for students.
   **Advanced Listings**: Support for "Combo Listings" (e.g., full semester kits) and "Countdown Timers" to drive urgency for graduating hostellers.
   **Real-Time Chat Hub**: Integrated, secure messaging via WebSockets—no need to exchange personal phone numbers.

3. 🗄️ **The Vault (Academic Ledger)**
   **Department-Wise Repositories**: Structured, permanent file organization replacing temporary group chats.
   **Resource Archiving**: Upload and access PYQs (Past Year Questions), lecture notes, and assignments.
   **Crowdsourced Curation**: An Upvote/Downvote system that surfaces premium study material and buries outdated content.

4. 👤 **The Profile (Gamified Identity)**
   **Karma System**: A reputation metric rewarding positive community contributions (e.g., high-voted notes, successful returns in Lost & Found).
   **Trust Badges**: High Karma translates to platform visibility and "Trusted Peer" badges.

# 🚀 Getting Started

To get a local copy up and running, follow these steps.

**Installation**

1. Clone the repo

```bash
git clone https://github.com/dilawarzAlgorithm/campus-square.git
```

2. Navigate to the directory

```bash
cd campus-square
```

3. Install dependencies

```bash
flutter pub get
```

4. Run the app

```bash
flutter run
```
